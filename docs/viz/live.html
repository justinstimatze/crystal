How the rules get rebuilt without a human: when commands start missing the rules (rule fell behind), two small models each label the new commands. Where they agree, that's the answer key; where they disagree we skip it (a stronger model can tie-break). A new rule is written from those labels and only kept if it passes a check — then it serves the commands that used to need the model.
